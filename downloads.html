Linux

Download the tarball here: spinframework-1.0.0.tar.gz

Or, to get the most recent package for Ubuntu, add some repositories to your system:

sudo apt-add-repository ppa:sat-metalab/metalab
sudo apt-add-repository ppa:flexiondotorg/flightgear

Then install the package:

sudo apt-get install spinframework

For more details, read the Quick Start guide.
